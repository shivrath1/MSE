# Activity 1 - Basic Math Calculator (OOP)

## Course
MSE800

## Week
Week 2

## Description
It performs basic mathematical operations and demonstrates the use of bject-oriented programming (OOP) concepts such as classes, methods and with functions. Developed in python

---

## Concepts Used
- Functions
- Classes and Objects
- Methods
- Exception Handling
- User Input Handling

---

## Features
- Supports basic operations: `+`, `-`, `*`, `/`, `%`
- Supports factorial calculation (`!`)
- Accepts:
  - Integers
  - Floating-point numbers
  - Complex numbers (e.g., `2+3j`)
- Handles invalid inputs and errors

---

## Project Structure

- **Functions**
  - `show_instructions()` - Displays usage instructions
  - `parse_input()` - Converts user input into correct data type

- **Class: Calculator**
  - `basic_ops()` - Performs arithmetic operations
  - `factorial()` - Calculates factorial of a number
  - `mathOps()` - Handles user interaction and executes operations
